# Chat-Forestfull – 관리자 UI 설계

## 1. 목적
- 관리자 전용 기능 제공
- 사용자 관리, 채팅 모니터링, 공지 발행, 결제/포인트 관리
- 간단한 템플릿 기반 구현 (Thymeleaf)

## 2. URL 구조 예시
- `/admin/dashboard` – 메인 대시보드
- `/admin/users` – 사용자 목록 및 권한 관리
- `/admin/users/{id}` – 사용자 상세 페이지 (강퇴, mute, 임시 매니저 권한 발급/회수)
- `/admin/chat` – 실시간 채팅 모니터링
- `/admin/notifications` – 공지 관리
- `/admin/payments` – 결제 내역 및 포인트 관리
- `/admin/logs` – 시스템 로그 및 감사 기록
- `/admin/settings` – 관리자 설정 (선택)

## 3. 기능

### 3.1 사용자 관리
- 가입 승인 / 접근 제한
- 경고, 강퇴, mute 기능
- 임시 매니저 권한 발급/회수
- 사용 기록 조회

### 3.2 채팅 모니터링
- 실시간 메시지 스트림 확인
- 금칙어 필터링
- 악성 메시지 감지
- 채팅 삭제/관리 기록 남김

### 3.3 공지 관리
- 상단 배너 / 채팅 상단 / 모달 팝업 발행
- 중요도 설정 (normal / high)
- 신규 공지 작성, 수정, 삭제

### 3.4 결제 및 포인트 관리
- 결제 로그 확인
- 포인트 구매 순위 확인
- 결제 취소/환불 상태 확인 및 권한 회수
- 감사 로그 기록

### 3.5 시스템 로그
- 임시 매니저 사용 내역
- 채팅 관리 작업 기록
- 서버 이벤트 기록
- 에러 및 Webhook 처리 로그

## 4. UI 설계 포인트
- 간단하고 직관적인 템플릿 기반 화면
- 실시간 데이터 반영 (WebSocket 또는 SSE)
- 데이터 테이블에 필터/정렬 기능
- 모달 사용으로 중요 작업 확인
- 관리자 전용 접근 권한 적용
